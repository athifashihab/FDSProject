# Residential Energy Forecasting  This project compares ARIMA, SARIMA, Prophet, XGBoost, LSTM, and GRU models for forecasting hourly residential energy consumption.  ## Models Used - ARIMA / SARIMA - Prophet - XGBoost - LSTM / GRU  ## Evaluation Metrics MAE, RMSE, MAPE, R²  ## Dataset California 2018 ResStock dataset (15-minute resolution)
